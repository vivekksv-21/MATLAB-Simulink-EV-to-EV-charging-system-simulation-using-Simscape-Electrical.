# EV-to-EV Charging System Simulation using MATLAB Simulink

## Overview

This project demonstrates energy transfer between two electric vehicle battery packs using MATLAB Simulink and Simscape Electrical.

The simulation models:

* Donor EV battery
* Receiver EV battery
* Controlled charging current
* Battery charge transfer
* Current monitoring
* Charge monitoring

## Model Architecture

Donor Battery → Controlled Current Source → Receiver Battery

## Simulation Results

The simulation successfully demonstrates:

* Energy transfer between batteries
* Donor battery charge reduction
* Receiver battery charge increase
* Controlled charging current operation

## Tools Used

* MATLAB R2025a
* Simulink
* Simscape Electrical
